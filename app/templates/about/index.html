Dataset Generation Algorithm

The platform derives a deterministic seed from the group leader's full name using SHA-256 hashing. Data is generated through a causal chain: Study Hours → Attendance → Homework Score → Exam Score, each step positively influenced by the previous with small Gaussian noise.

Orientation is never random. It is computed as a weighted academic index — Exam Score (60%), Homework Score (25%), Attendance (15%) — with students at or above the cohort median classified as Scientific and below as Literary.

Statistics Concepts

Question 1 applies descriptive statistics: mean, median, mode, variance, quartiles, skewness, kurtosis, and IQR-based outlier detection.

Question 2 uses Pearson/Spearman correlation and multiple linear regression with R² evaluation.

Machine Learning

Question 3: K-Means clustering with elbow method and silhouette score for optimal K selection.

Question 4: Decision Tree, Random Forest, KNN, and Logistic Regression compared via accuracy, precision, recall, and F1.
